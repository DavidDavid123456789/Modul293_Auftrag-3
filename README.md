# Dave's Burger – README

## Projektübersicht

Dave's Burger ist eine responsive Restaurant-Webseite mit mehreren Unterseiten.  
Die Seite wurde mit HTML, CSS und SCSS erstellt.

Ziel des Projekts war es, eine moderne und übersichtliche Webseite für ein Fast-Food Restaurant zu entwickeln.

---

## Verwendete Technologien

- HTML5  
- CSS3  
- SCSS / Sass  
- Responsive Design  
- Flexbox  
- CSS Grid  
- GitHub Pages

---
